# ESP32-RPi-LCD-demo-mandelbrot-raytrace

## Pinouts:
                   ESP32         RPI LCD             
                     19          TFT_MISO
                     23          TFT_MOSI
                     18          TFT_SCLK
                     15          TFT_CS
                      2          TFT_DC
                      4          TFT_RST
                     22          TOUCH_CS 

## Demos:                             
  Amstrad CPC bootscreen with custom font           
  Mandelbrot                           
  Raytraced Scene                        
     
## Typical Benchmark Speed:
  Mandelbrot: 4 seconds Raytrace: 13 seconds    
     
## Libs
  Uses TFT_eSPI
